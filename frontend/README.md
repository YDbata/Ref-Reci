# Front-End

#### 반응형 SPA Website & Web App

> 작성자: 박지우

### 기능

- 회원가입 및 로그인
- 냉장고 DB 조회 및 갱신
  - 식품 유통기한 달력 UI로 확인
  - 기기로 입력한 수량, 유통기한 수정 및 식재료 삭제
  - 식재료 바구니 담기
- 레시피 DB 조회 및 즐겨찾기
  - 바구니에 담은 식재료를 바탕으로 한 레시피 조회
  - 최신 순, 즐겨찾기 많은 순 레시피 조회
